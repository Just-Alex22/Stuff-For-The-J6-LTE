# Why does this repo exist?

The Samsung Galaxy J6 LTE (formerly samsung-j6lte) is a pretty niche device in the custom romming community. So it's pretty difficult to find ROMs and tools that actually work for this device.

This repo is just a collection of useful tools and ROMs that can be used to ROM this device. Any suggestions are welcome, just open a Pull Request or an issue, and it will be added in no time.

#### Which model is the right one for me?
Just go to settings and go to the **About Phone** screen, your specific model should appear right away.

If you can't access settings for whatever reason, just turn your phone around and check the bottom part. You should see a small text with some technical information; your specific number should be on the first line.

#### Why do I need a custom recovery?
Custom recoveries are necessary to properly flash custom ROMs, wipe partitions, sideload, etcetera. Without one, the process can´t continue.
